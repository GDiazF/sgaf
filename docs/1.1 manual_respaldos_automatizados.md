# Guía Maestra: Configuración de Respaldos SGAF (Ubuntu a Google Drive)

Este manual detalla cómo se configuró el sistema de respaldos desde cero. Úsalo como referencia para reconfigurar el servidor o migrarlo.

## 1. Instalación de Herramientas
Primero, asegúrate de estar conectado por SSH al servidor y ejecuta:
```bash
sudo apt update
sudo apt install rclone -y
```

## 2. Vinculación con Google Drive (Rclone)
Para conectar el servidor con la cuenta de Google Drive:

1. Ejecuta: `rclone config`
2. Elige **`n`** (New remote) y nómbralo como: **`drivesoporteti`** (es importante usar este nombre exacto).
3. Selecciona el tipo: **`drive`** (Google Drive).
4. *Client ID* y *Client Secret*: **Presiona Enter** en ambos para dejarlos en blanco.
5. *Scope*: Elige **`1`** (Acceso total).
6. *Service Account file*: **Presiona Enter** para dejarlo en blanco.
7. *Edit advanced config?*: Escribe **`n`** y presiona Enter.
8. **IMPORTANTE** - *Use auto config?*: Escribe **`n`** (No), porque estás por SSH sin navegador.
9. Rclone te dará un comando largo. Abre PowerShell en Windows y ejecuta: 
   `.\rclone.exe authorize "drive" "clave_larga..."`
   Autoriza en el navegador de Windows y copia el código JSON resultante.
10. Pega ese código de vuelta en la terminal de Ubuntu.
11. *Configure this as a Shared Drive?*: Escribe **`n`** y presiona Enter.
12. *Keep this remote?*: Escribe **`y`** y presiona Enter.
13. Escribe **`q`** para salir del asistente.

## 3. 🛡️ Creación del Script de Respaldo
El script debe ubicarse en tu carpeta de usuario: `/home/tu_usuario/sgaf/scripts/respaldo_automatico_sgaf.sh`.

> [!WARNING]
> ⚠️ **ATENCIÓN AL CAMBIAR SERVIDOR:** Dentro del código de este script, existen 2 líneas al inicio (`BACKUP_DIR` y `MEDIA_DIR`). Asegúrate de editarlas para que apunten a tu nombre de usuario real (ej. `/home/slepiquique/...`).

### A. Respaldo de Base de Datos (Copia)
- Genera un archivo `.sql` desde el contenedor `sgaf_db_prod`.
- Comprime el archivo con `gzip`.
- Sube el archivo a Drive usando: `rclone copy drivesoporteti:SGAF_Backups`
- **Rotación**: Borra archivos locales y en la nube con más de 14 días.

### B. Respaldo de Media (Sincronización)
- Sincroniza la carpeta física de fotos y documentos.
- Usa el comando: `rclone sync drivesoporteti:SGAF_Media`
- **Ventaja**: Solo sube archivos nuevos o modificados.

Para darle permisos de ejecución al script:
```bash
chmod +x ~/sgaf/scripts/respaldo_automatico_sgaf.sh
```

## 4. 🤖 Automatización con Cron
Para que el servidor trabaje solo, se programó el archivo `crontab`.

1. Entra al editor: `crontab -e`
2. 📌 **¡CRÍTICO AL MIGRAR!** Agrega estas líneas al final (Asegúrate de modificar `tu_usuario` por tu usuario real del sistema, por ejemplo `slepiquique`):
```bash
# Respaldos diarios: 9:00 AM y 6:00 PM
0 9 * * * /home/tu_usuario/sgaf/scripts/respaldo_automatico_sgaf.sh >> /home/tu_usuario/backup_log.txt 2>&1
0 18 * * * /home/tu_usuario/sgaf/scripts/respaldo_automatico_sgaf.sh >> /home/tu_usuario/backup_log.txt 2>&1
```

## 5. Verificación
- Ver manual: `cat ~/backup_log.txt`
- Ejecución manual: `bash ~/sgaf/scripts/respaldo_automatico_sgaf.sh`

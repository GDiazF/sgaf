# Guía Maestra: Configuración de Respaldos SGAF (Ubuntu a Google Drive)

Este manual detalla cómo se configuró el sistema de respaldos desde cero. Úsalo como referencia para reconfigurar el servidor o migrarlo.

## 1. Instalación de Herramientas
Primero, asegúrate de estar conectado por SSH al servidor y ejecuta:
```bash
sudo apt update
sudo apt install rclone -y
```

## 2. Vinculación con Google Drive (Rclone)
Para conectar el servidor con la cuenta de Google Drive:

1. Ejecuta: `rclone config`
2. Elige **n** (New remote) y nómbralo como: **`drivesoporteti`**
3. Selecciona el tipo: **`drive`** (Google Drive).
4. Deja en blanco *Client ID* y *Client Secret*.
5. Scope: Elige **1** (Acceso total).
6. **Importante**: Cuando pregunte *Use auto config?*, selecciona **n** (No), porque estás por SSH.
7. Rclone te dará un comando para ejecutar en tu PC local (Windows):
   - Abre PowerShell en Windows y ejecuta: `./rclone authorize "drive" "clave_que_te_dio_remoto"`
   - Autoriza en el navegador y copia el código JSON resultante.
8. Pega ese código de vuelta en la terminal de Ubuntu.
9. Finaliza el asistente confirmando con **y**.

## 3. Creación del Script de Respaldo
El script se encuentra en `/home/svtest/sgaf/scripts/respaldo_automatico_sgaf.sh`.

### A. Respaldo de Base de Datos (Copia)
- Genera un archivo `.sql` desde el contenedor `sgaf_db_prod`.
- Comprime el archivo con `gzip`.
- Sube el archivo a Drive usando: `rclone copy drivesoporteti:SGAF_Backups`
- **Rotación**: Borra archivos locales y en la nube con más de 14 días.

### B. Respaldo de Media (Sincronización)
- Sincroniza la carpeta física de fotos y documentos.
- Usa el comando: `rclone sync drivesoporteti:SGAF_Media`
- **Ventaja**: Solo sube archivos nuevos o modificados.

Para darle permisos de ejecución al script:
```bash
chmod +x ~/sgaf/scripts/respaldo_automatico_sgaf.sh
```

## 4. Automatización con Cron
Para que el servidor trabaje solo, se programó el archivo `crontab`.

1. Entra al editor: `crontab -e`
2. Agrega estas líneas al final:
```bash
# Respaldos diarios: 9:00 AM y 6:00 PM
0 9 * * * /home/svtest/sgaf/scripts/respaldo_automatico_sgaf.sh >> /home/svtest/backup_log.txt 2>&1
0 18 * * * /home/svtest/sgaf/scripts/respaldo_automatico_sgaf.sh >> /home/svtest/backup_log.txt 2>&1
```

## 5. Verificación
- Ver manual: `cat ~/backup_log.txt`
- Ejecución manual: `bash ~/sgaf/scripts/respaldo_automatico_sgaf.sh`

# Manual de Preparación del Servidor (Ubuntu) - SGAF

Este documento detalla los pasos para dejar un servidor Ubuntu limpio listo para hospedar el sistema SGAF mediante contenedores Docker.

## 1. Acceso y Seguridad Inicial
- **Servidor**: Ubuntu (Instalación Limpia)
- **Acceso Directo**: Se debe instalar el servidor SSH si no está presente:
  ```bash
  sudo apt update
  sudo apt install openssh-server -y
  sudo ufw allow ssh
  ```

## 2. Instalación de Docker y Herramientas
Se optó por la instalación mediante los repositorios oficiales de Ubuntu para mayor compatibilidad:

### Instalar Docker Engine
```bash
sudo apt update
sudo apt install docker.io -y
```

### Instalar Docker Compose (V2)
```bash
sudo apt install docker-compose-v2 -y
```

### Configurar Permisos de Usuario
Para operar Docker sin requerir `sudo` en cada comando:
```bash
sudo usermod -aG docker ${USER}
# Requiere cerrar y volver a abrir sesión SSH
```

## 3. Instalación de Git
Necesario para la descarga y actualización del código:
```bash
sudo apt install git -y
```
